# Job-Application-Management-System
Created a full-stack application of a database system for managing job applications.

Implemented the user interface using Handlebars.js, HTML, CSS, Bootstrap.

Implemented the REST API using Node.js, Express.js.

Implemented the database server using SQL/MySQL.
